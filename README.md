# MultiversX Blockchain School

Course materials for the MultiversX Blockchain School program.

## Structure

- **lessons/** — slide sources, built PDFs/PPTXs, and homework assignments
- **lessons/source/** — Marp Markdown slide decks
- **lessons/pdf/** / **lessons/pptx/** — exported presentation files
- **homework.md** — homework assignments for students

## Homework

Assignments are listed in [lessons/homework.md](lessons/homework.md). Submissions are verified on-chain on the MultiversX Devnet.

## Building Slides

```bash
cd lessons
./build-slides.sh
```
